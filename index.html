CSS

はじめに

同じファイル内にCSSを書き込む場合
<head>タグ内に<style>を書き、その中にCSSコードを書き込んでいく
別ファイルにCSSを書き込む場合
  1. <head>タグ内にlink:cssと入力し、<link rel="stylsheet" href="style.css">を生成する。
  2. 上記のhref="style.css"にマウスオーバーしリンク先を表示をクリック、
    ファイルの作成をクリックすることでstyle.cssファイルが作成される。
    視認性のため、タブをhtml横に配置するようドラッグすると良し。
  3. 作成されたstyle.cssファイル内冒頭に@charset "UTF-8";と入力し、CSS内の文字コードを指定する。
    （コード前に空白がないよう注意）
  4. 以降、CSSのスタイリングをこのファイルに書き込むことで、HTMLファイルで情報が反映されるようになる
CSSにコメントを書き込む場合
コマンドは同じくcommamd + /を使用するが、HTML時<!-- -->と異なり、/* */と表記される

CSSスタイリング指定用語

要素 名称 役割
p セレクター スタイリングしたい要素を表す
{ } 宣言ブロック スタイリングを指定するためのブロック
background-color: pink; 宣言 セレクターに対しどのようなスタイリングを行うかを指定する要素。
宣言ブロック内にいくつも書き込める
background-color プロパティ どのようなスタイリングをするかを宣言するコード。末尾は:で区切る
pink 値（プロパティの値） プロパティで宣言する値を指定するコード。末尾は;で区切る

ボックスモデル

CSSでスタイリングを行う際によく使われる領域ごとの用語説明

要素 名称 役割
content box コンテントボックス HTMLの要素を配置したときに確保される領域。
background-colorなどで視認できる領域。
border box ボーダーボックス borderというプロパティで配置できる、content boxの周囲にできる境界線までを含めた領域。
padding box パディングボックス paddingプロパティを使うことで、border boxの内側につけることができる余白の領域を指す。
margin box マージンボックス padding boxに対し、border boxの外側につける余白の領域を指す。marginプロパティを用いる。

一括指定プロパティルール

プロパティのサイズをtop,right,botttom,leftといったキーワードを使用せず指定する際のルール

プロパティ ルール
: 16px; 上下左右全てを16pxで返す
: 16px 8px; 上下を16px、左右を8pxで返す
: 16px 8px 4px; 上を16px、左右を8px、下を4pxで返す
: 16px 8px 4px 2px; 上を16px、右を8px、下を4px、左を2px　と時計回りで返す

marginの相殺

上下に隣接したmarginがある場合、値が大きい方のmarginのみ適用され
小さい値（同値の場合片方）が打ち消される仕様。

コード/プロパティ 名称 意味
background-color: ; バックグラウンドカラー 背景色を指定するプロパティ
border: ; ボーダー content boxの周囲にできる境界線までを含めた領域。
padding: ; パディング border boxの内側につけることができる余白。
margin: ; マージン border boxの外側につける余白。ブラウザにデフォルトでついている場合が多く、
ゼロからスタイリングする際にはmargin: 0;を宣言しリセットする必要がある
max-width: ;
min-width: ;		 マックスウィドゥス
ミンウィドゥス		 widthの値を％で指定した際、ページのサイズを変更した場合の
最大値（最小値）を設定するプロパティ
auto オート marginの左右の値のうち、余っている空間をautoが指定されている方に寄せる値。
margin: 0 auto 0 0;は右側に空間を、margin:0 0 0 auto;は左側に空間を、
margin:0 auto;(margin:auto;/margin:0 auto 0;も同義)では左右に50%ずつ空間が分けられる。
box-sizing: ; ボックスサイズ width/heightは基本的にcontent boxの値を指定するが、
このプロパティはwidth/heightの値がどの領域におけるサイズかを指定するかを変更する。
値にはボックスモデル内の領域を指定する。
overflow: hidden;
overflow: scroll;		 オーバーフロー ヒドゥン
スクロール		 コンテンツが領域をはみ出してしまう場合の制御プロパティ。
hiddenは領域をはみ出したコンテンツを隠す値。
scrollは、hiddenと同じく領域からはみ出したコンテンツを隠しつつ、
スクロールすることで領域内で視認できる値。
border-radius: ; ボーダーラディウス 領域の角を丸めるためのプロパティ。
値をpxで指定すると、指定した長さの半径を持つ円で丸める。
％で指定すると、”その領域の辺の長さ％”で丸くなるため、
正方形の領域でborder-radiuus: 50%;と指定すると、正円の領域ができる。
また、このプロパティも一括指定することができ、左上から時計回りに指定される。
box-shadow: ①px ②px; ボックスシャドウ 領域の周りに影をつけるプロパティ。値については、
①→左右にどれだけずらすか（+右方向　-左方向）、
②→上下にどれだけずらすか（+下方向　-上方向）を指定する。
さらに、3つ目の値を足すと影をどれだけぼかすかを指定する（大きい値ほどぼかしが大きい）。
fonntt-size: ; フォントサイズ 文字の大きさを指定するプロパティ。
<p>の文字サイズは基本的に16px。
font-weight: ; フォントウェイト フォントの太字を指定するプロパティ。
太字にする場合は値をbold;、解除する場合は値をnormal;にする。
text-align: ; テキストアライン 領域内でのテキストの配置揃えを指定するプロパティ。
右揃えはright;、左揃えはleft;、中央揃えはcenter;を指定する。
line-height: ; ラインハイト 行ボックス(<h1>や<p>などにテキストを配置するとできる行ごとの領域)の
高さを指定するプロパティ。
値をpxで指定すると、値から行ボックスのfont-sizeを引いて1/2した値が、
Harf Leadingと呼ばれるfont-sizeの上下にある領域のサイズとして反映される。
値に単位を入れず1.5などと指定すると、font-sizeの1.5倍が自動で配置される。
inherit インヘリット 大枠からのスタイルの継承(color,font-size,font-weight,text-aline等)がされないプロパティを、
強制的に継承させる値。borderを継承させる場合はborder: inherit;と指定する。
※プロパティが継承するかどうかは、MDNに記載があるので随時チェック。
font-family: ; フォントファミリー ブラウザ・デバイスごとに変わるフォントをある程度指定するためのプロパティ。
複雑な指定になり、フォントに関する知識も要するため、
慣れないうちは都度「fonnt-family　最新」と検索しながら身につける。
sans-serif 総称フォントファミリー fonnt-familyのプロパティで最後につける値。
「指定したフォントがない場合、そのデバイスにあるゴシック系のフォントを使う」指定
